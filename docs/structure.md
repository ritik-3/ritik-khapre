# Folder structure

## src/

- `app/` - application composition (root `App`)
- `components/` - reusable UI primitives (small, composable)
- `config/` - site content/config (name, links)
- `layout/` - layout building blocks (Header, etc.)
- `sections/` - page sections (Hero, Projects, About, Contact, Footer)
- `styles/` - global styles (Tailwind import, base resets)

## docs/

Project documentation and notes.
